# Shadadam GitHub Pages Site

Upload these files to the root of the `keeper121.github.io` repository.

Before publishing, replace:

- `pub-YOUR_PUBLISHER_ID` in `app-ads.txt`
- `Miklabs` in the copyright line if you want to publish under a different name

Expected URLs after GitHub Pages deploys:

```text
https://keeper121.github.io/
https://keeper121.github.io/support.html
https://keeper121.github.io/privacy.html
https://keeper121.github.io/terms.html
https://keeper121.github.io/app-ads.txt
```

Use these in App Store Connect:

```text
Support URL: https://keeper121.github.io/support.html
Marketing URL: https://keeper121.github.io/
Privacy Policy URL: https://keeper121.github.io/privacy.html
```
